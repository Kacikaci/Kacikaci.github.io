开发基于SSM的管理系统-1安装所需软件|java

macOS High Sierra 10.13.4

tomcat

  1. 从官网下载tomcat，选择tar.gz的安装包（在core一列中）。下载后解压，将解压出来的文件夹重新命名为“Tomcat”，拖入《Library》文件夹。（注意：不要拖入/Users/flower/Library，而是拖入/Library）

  2. Terminal
    cd /Library/Tomcat/bin

  3. 授权bin目录下的所有操作
    sudo chmod 755 *.sh
    输入密码，回车
    >chmod改变文件的存取模式：三个数字分别表示[User][Group][Others]，读4写2执行1。参数-R表示递归改变当前目录下的所有目录和文件的权限。eg.chmod -R 777 foldername

  4. sudo sh ./startup.sh

  5. 浏览器输入http://localhost:8080/

  6. 关闭Tomcat
    sh ./shutdown.sh
    回车

  7. cd /Library/Tomcat/bin
    sh ./version.sh # Eclipse

  8. 发现原本安装的Eclipse不是For JavaEE，表现为没有新建Dynamic Web Project的选项，所以先重新安装Eclipse。

  9. 下载时点击Select Another Mirror 大连东软信息学院。Download eclipse-inst-jre-mac64.dmg

  10. 双击安装程序，选择Eclipse IDE for Enterprise Java and Web Developers。
    > Java11+VM /Library/Java/JavaVirtualMachines/jdk-12.0.1.jdk/Contents/Home
    Installation Folder Users/flower/eclipse/jee-2021-122
    WorkPlace /Users/flower/eclipse-workspace
    # MySQL

  11. 下载MySQL安装包，注意和macOS版本匹配，我的macOS没有更新到最新的版本，所以不能下载最新版本的MySQL安装包，一开始下载错了，重新下载了8.0.22版本的。

  12. 查看MySQL路径
    cd /Usr/local/mysql
    ls

  13. 添加MySQL的路径
    cd ~
    touch .bash_profile
    open -e .bash_profile

  14. 添加一行export PATH="/usr/local/mysql/bin:$PATH"

  15. 保存，然后查看有没有保存成功
    source ~/.bash_profile
    echo $PATH

  16. 验证MySQL的运行
    重新打开终端mysql -u root -p输入密码

  17. 可以下载MySQLWorkbench数据库管理工具

Hexo支持LaTex公式

问题：如何让Hexo的blog页面正常显示LaTex公式

解决方法：

1、npm uninstall hexo-math --save

2、npm install hexo-renderer-mathjax --save

3、npm uninstall hexo-renderer-marked --save

4、npm install hexo-renderer-kramed --save

5、修改/yourBlogFolder/node_moudles/hexo-renderer-mathjax/mathjax.html

修改的是src中http......latest.js这一部分的内容，具体改成什么地址参见这里的NEWS页面

6、修改主题配置文件（themes>nest>_config.yml）

7、在需要LaTex渲染的post的Front-matter里写上mathjax: true

Hexo页面不更新

问题：执行hexo g -d 后，github仓库更新了，但是blog页面没有更新

解决方法：

  1. git remote add origin https://github.com/Kacikaci/Kacikaci.github.io.git
  2. git remote -v
  3. git remote rm Cuiwen

总的来说就是，只留下origin

或者-

也有可能是因为一篇post的标题太长或者标题中出现了“|”等符号，导致没有办法更新页面。这种时候，标题取简单一点，并且不要出现特别的符号就可以了。
