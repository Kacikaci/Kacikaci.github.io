Hexo支持LaTex公式

Posted on 2019-11-07 Edited on 2019-11-11 In 功能与技巧

问题：如何让Hexo的blog页面正常显示LaTex公式

解决方法：

1、npm uninstall hexo-math --save

2、npm install hexo-renderer-mathjax --save

3、npm uninstall hexo-renderer-marked --save

4、npm install hexo-renderer-kramed --save

5、修改/yourBlogFolder/node_moudles/hexo-renderer-mathjax/mathjax.html

修改的是src中http……latest.js这一部分的内容，具体改成什么地址参见这里)的NEWS页面

6、修改主题配置文件（themes>nest>_config.yml）

7、在需要LaTex渲染的post的Front-matter里写上mathjax: true
