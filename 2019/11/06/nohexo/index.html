Hexo页面不更新

问题：执行hexo g -d 后，github仓库更新了，但是blog页面没有更新

解决方法：

  1. git remote add origin https://github.com/Kacikaci/Kacikaci.github.io.git
  2. git remote -v
  3. git remote rm Cuiwen

总的来说就是，只留下origin

或者-

也有可能是因为一篇post的标题太长或者标题中出现了“|”等符号，导致没有办法更新页面。这种时候，标题取简单一点，并且不要出现特别的符号就可以了。
